# UserHub Dashboard

Angular User Management Dashboard with dynamic user operations and role analytics.

## Live Demo
https://userhubdashboard.netlify.app

## Features
- Add User
- Delete User
- Search User
- Pagination
- Dynamic Pie Chart
- Role Distribution
- Responsive UI
- Chart.js Integration

## Tech Stack
Angular  
TypeScript  
Chart.js  
HTML  
CSS  
RxJS  

## Run Locally

npm install

ng serve

Open:
http://localhost:4200
